# Code of Conduct

## Our pledge

We want this project to be useful, respectful, and practical for founders and operators.

## Expected behavior

- Be respectful and specific.
- Focus feedback on the work.
- Assume sample data is synthetic unless clearly documented otherwise.
- Help keep the project usable for non-technical users.

## Unacceptable behavior

- Harassment, threats, or personal attacks.
- Publishing private customer, employee, investor, or company data.
- Adding misleading claims about production usage.

## Enforcement

Maintainers may remove comments, issues, or contributions that violate this code of conduct.
