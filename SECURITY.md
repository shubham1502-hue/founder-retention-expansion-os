# Security

## Supported versions

This public template is maintained on the `main` branch.

## Reporting a vulnerability

Open a GitHub issue if the report does not include sensitive information. If it includes sensitive data, contact the maintainer privately.

## Data handling

This repo is designed for synthetic sample data and private internal forks.

Do not commit:

- private customer data
- private prospect data
- private employee data
- private investor data
- API keys
- access tokens
- credentials
- confidential account notes

The base workflow does not require paid APIs or external services.
