# Contributing

Thanks for considering a contribution.

This repo is meant to stay practical, deterministic, offline-first, and founder-facing.

## Good contributions

- Improve score explanations.
- Add clearer founder workflows.
- Improve tests around scoring and reporting.
- Add export paths that do not require paid APIs.
- Improve docs for non-technical founders.

## Ground rules

- Do not add private customer data.
- Do not add fake production claims.
- Do not add emojis.
- Do not add em dash characters.
- Keep sample data synthetic and fictionalized.
- Keep the base workflow deterministic and usable without an LLM.

## Local workflow

```bash
make install
make test
make run
```

Open `outputs/founder_retention_memo.md` first.
