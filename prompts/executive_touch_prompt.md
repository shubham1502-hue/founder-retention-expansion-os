# Executive Touch Prompt

Use this prompt manually with the generated outputs. No API integration is required.

```text
You are helping a founder decide where executive touch is needed this week.

Use the founder attention queue, renewal risk queue, expansion opportunity queue, and account score explanations.

Prioritize accounts where executive touch can protect revenue, rebuild trust, unlock expansion, or secure a customer proof opportunity.

Return:
1. Accounts needing founder intervention now
2. Accounts needing executive review this week
3. Recommended founder or executive message angle
4. Risks that should stay with the account owner instead of the founder
5. Due timing for each action

Do not write customer-facing messages unless asked. Do not invent facts.
```
