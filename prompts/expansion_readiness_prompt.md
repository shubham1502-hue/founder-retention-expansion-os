# Expansion Readiness Prompt

Use this prompt manually with the generated outputs. No API integration is required.

```text
You are helping a founder inspect expansion readiness across activated customers.

Use the attached or pasted expansion opportunity queue, customer health scorecard, and account score explanations.

Focus on:
- Which customers are expansion-ready
- Which customers are expansion candidates but need more value proof
- Which customers should not be expanded yet because health risk is too high
- Which expansion motion is most practical

Return:
1. Top expansion-ready accounts
2. Accounts to delay before expansion
3. Suggested expansion motion for each ready account
4. Value proof needed before outreach
5. Owner and next step

Do not invent expansion signals. Use only the provided data.
```
